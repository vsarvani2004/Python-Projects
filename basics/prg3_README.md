# 💰 Expense Tracker (CLI)

## 📌 Description

A simple command-line based Expense Tracker built using Python to help users record and manage their daily expenses.

---

## 💡 Problem

People often lose track of their daily spending, which makes it difficult to manage finances effectively.

---

## ✅ Solution

This program allows users to:

* Add expenses with amount and category
* View total spending
* View all recorded expenses

---

## ⚙️ Features

* Add new expenses
* View total expense amount
* View all expenses with category
* Simple menu-driven interface
* Handles empty data cases

---

## ▶️ How to Run

```bash
python expense_tracker.py
```

---

## 📥 Input Format

User interacts through menu:

```
1. Add Expense
2. View Total Expense
3. View All Expenses
4. Exit
```

Example:

```
Enter your choice: 1
Enter amount: 200
Enter category: food
```
```
Enter your choice: 1
Enter amount: 500
Enter category: travel
```

---

## 📤 Output Format

```
Expense added successfully!

Total Expense: 700

Your Expenses:
1. Amount: 200, Category: food
2. Amount: 500, Category: travel
```

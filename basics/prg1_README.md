# 🧮 Simple Calculator

## 📌 Description

A basic Python calculator that performs arithmetic operations.

## ⚙️ Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero
* Accepts decimal numbers

## ▶️ How to Run

```bash
python calculator.py
```

## 📥 Input Format

* Enter first number
* Enter operator (+, -, *, /)
* Enter second number

## 📤 Output Format

Displays the result of the operation

## 💡 Example

Enter your number1: 10
Enter your operator: +
Enter your number2: 5
Result: 15

# 🎓 Student Grade Calculator

## 📌 Description

A Python program that calculates the total marks, average, and grade of a student based on multiple subject scores.

---

## 💡 Problem

Manually calculating average marks and assigning grades can be time-consuming and error-prone.

---

## ✅ Solution

This program:

* Takes marks for multiple subjects
* Calculates total and average
* Assigns grade based on performance
* Displays pass/fail status

---

## ⚙️ Features

* Supports multiple subjects
* Calculates total and average marks
* Assigns grades (A+, A, B, C, D, F)
* Displays pass/fail status
* Validates input (marks must be between 0 and 100)

---

## ▶️ How to Run

```bash
python grade_calculator.py
```

---

## 📥 Input Format

* Enter total number of subjects
* Enter marks for each subject

Example:

```
Enter total number of subjects: 3
Enter marks for subject 1: 85
Enter marks for subject 2: 90
Enter marks for subject 3: 78
```

---

## 📤 Output Format

```
Total Marks: 253
Average Marks: 84.33
Grade: A
Status: Pass
```

---

## 🧠 Grading Criteria

* 90 and above → A+
* 80–89 → A
* 70–79 → B
* 60–69 → C
* 50–59 → D
* Below 50 → F
